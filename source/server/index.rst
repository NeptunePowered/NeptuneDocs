=================
Creating a Server
=================

Overview
========

What is this about?
~~~~~~~~~~~~~~~~~~~

The articles in this section explain how to create and run a server powered by Neptune.

.. note::

    The provided instructions are for Neptune. CanaryMod has their own `documentation <http://canarymod.net/books/>`__ for their implementation.

Who should read this?
~~~~~~~~~~~~~~~~~~~~~

This section is intended for users who wish to create their own server using Neptune. The user can be a server owner or plugin developer.
