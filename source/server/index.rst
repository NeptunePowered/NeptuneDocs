=================
Creating a Server
=================

Overview
========

What is this about?
~~~~~~~~~~~~~~~~~~~

The articles in this section explain how to create and run a server powered by Neptune.

.. note::

    The instructions provided, will be for Neptune. CanaryMod has their own `documentation <http://canarymod.net/books/>`__ for their implementation.

Who should read this?
~~~~~~~~~~~~~~~~~~~~~

This section is intended for users who wish to create a new server using Neptune. This could be a server admin, or a plugin developer.
